# 健康脉络 (HealthTrace) - 个人体检健康趋势深度分析系统

这是一个基于 Python 的本地数据处理与可视化分析工具，可以一键读取您历年的 PDF 体检报告，智能提取各年度的核心健康指标（如 BMI、血糖、血脂、肝功能等），并通过预设算法进行趋势评分，最后生成一个高度可交互且设计精美的动态前端页面（仪表盘），以直观地展现各项体检数据的历年演变。

## 项目目录结构

```text
📁 体检报告项目根目录/
  ├── 📄 一键生成报告.bat     # 一键自动化脚本（双击使用）
  ├── 📄 README.md            # 项目说明文档
  ├── 📄 prompt.md            # 本项目原始的设计需求提示词
  ├── 📁 report/              # ⭐ 请把您的所有 PDF 体检报告放入此处！
  ├── 📁 src/                 # 所有的 Python 源代码存放处
  │    ├── parser.py          # PDF 解析与体检项目智能提取脚本
  │    ├── analyze_health.py  # 风险评估与健康评分计算脚本
  │    └── generate_dashboard.py # 生成可视化 HTML 页面的渲染脚本
  ├── 📁 data/                # 生成的中间结构化数据
  │    ├── health_data.json   # 从 PDF 中提取出的原始格式化数据
  │    └── chart_data.json    # 转换后用于图表渲染的时间序列数据
  ├── 📁 docs/                # 生成的文本报告
  │    └── health_report.md   # AI 深度的历年体检诊断与综合建议文档
  └── 📁 output/              # 生成的成果物
       └── dashboard.html     # ✨ 最终生成的分析仪表盘网页（可双击打开）
```

## 准备工作

由于脚本使用了如 `pdfplumber`, `pandas` 等数据处理库，若您需要在您的 Python 环境中运行，确保项目根目录存在 `venv` 虚拟环境，并且已经安装了相关的第三方库：

```bash
pip install pdfplumber pandas
```
*(本项目已在本地配置了 `venv`，如果您是在原始电脑上使用，直接双击一键脚本即可，无需再执行环境安装。)*

## 使用方法

日常更新体检数据时，操作非常简单：

1. **添加报告**：将您从医院获取的**标准 PDF 体检报告**放入项目根目录下的 `report` 文件夹。
2. **运行脚本**：双击运行根目录下的 `一键生成报告.bat` 脚本。
   *(如果您使用的是 Mac 或 Linux，可以打开终端并在根目录下依次运行 `python src/parser.py`, `python src/analyze_health.py` 以及 `python src/generate_dashboard.py`)*
3. **查看结果**：脚本运行成功后，去 `output` 文件夹里找到 `dashboard.html`，双击在浏览器中打开，即可查阅您最新的健康分析报告和动态折线图！

## 功能亮点

- **自动纠错与同义词映射**：各医院的指标称呼有时不同（如“空腹血糖”和“葡萄糖”），脚本已内置常见医院同义词映射词典，自动标准化名称。
- **缺失数据容错**：如果在某年没测特定的项目，图表的折线将自动断开以体现真实的采样轨迹，而不会让数据突变成 0 导致误判。
- **健康打分与风险预警**：根据各项临床参考值（如 BMI 24+，血脂高等）对指标进行分级评判，在右侧文本报告中提示您优先关注的Top3核心风险。
- **现代化科技感排版**：仪表盘完全采用基于 Tailwind CSS 生成的深色星空玻璃拟物化设计 (Glassmorphism) 以及 ECharts，无需连接互联网服务器，直接本地打开体验一流效果！
